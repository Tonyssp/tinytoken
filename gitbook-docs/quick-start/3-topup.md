# (3) เติมเครดิต

## เติมเครดิต

**Quick Start** · 2026/6/8 · อ่านไม่เกิน 1 นาที

หน้าสั้น ๆ สำหรับการเติมเครดิตและดูยอดคงเหลือ

## ขั้นตอนเติมเครดิต

- เปิดหน้า Wallet หรือหน้าความคืบหน้าการเติมเงิน
- เลือกจำนวนเงิน แล้วรอ QR หรือข้อมูลโอนเงินปรากฏ
- ชำระเงินให้ตรงกับยอดที่เลือก และรอระบบอัปเดตยอดคงเหลือ

> 
ถ้าหน้าเติมเงินของคุณเปลี่ยนแปลงภายหลัง คุณสามารถเปลี่ยนตัวอย่างและรูปประกอบในหน้านี้ได้ทีละขั้น
